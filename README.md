### Currently building

🏥 **[Benefit Plan Standard](https://benefitplanstandard.org)** — An open-source, vendor-neutral JSON schema for normalizing health plan data. Filling the gap between raw carrier PDFs and structured, interoperable benefit information.

🔬 **Healthcare data infrastructure** — Turning unstructured benefit documents into normalized, field-level-traceable data. Glass Box provenance (every extracted value cites its source page and location).

📐 **FHIR community contributor** — Engaged with HL7 Financial Management workgroup. BPS is complementary to FHIR, handling the upstream normalization problem that FHIR R6 `InsurancePlan` doesn't cover.
